# Claude Code Demo

A hands-on guide and demo repository for understanding and using [Claude Code](https://code.claude.com) — Anthropic's agentic coding tool.

---

## Getting Started

Start with **[The Building Blocks of Claude Code](The%20Building%20Blocks%20of%20Claude%20Code.md)** — a comprehensive guide that breaks down every core concept of Claude Code, including:

- The conversation loop and how Claude "thinks"
- Tools, commands, and skills
- Subagents and agent teams
- Memory, MCP servers, hooks, and plugins
- Safety systems and best practices

This guide is written for business leaders, product managers, and anyone curious about how Claude Code works under the hood.

## Demo: Earnings Reports

The [Demo/](Demo/) folder contains sample outputs generated by the `/earnings` Claude Code command and skill, showcasing what Claude Code can produce in practice:

| File | Description |
|------|-------------|
| [AAPL_Q4_FY2025_Earnings_Update.md](Demo/AAPL_Q4_FY2025_Earnings_Update.md) | Apple Q4 FY2025 earnings analysis report |
| [Apple_Q1_FY2026_Earnings_Update.md](Demo/Apple_Q1_FY2026_Earnings_Update.md) | Apple Q1 FY2026 earnings analysis report |
| [charts/](Demo/charts/) | Generated charts (revenue, EPS, margins, segments, valuation bands, and more) |

The demo also includes notes on how the reports were created:

- [Commands Demo.txt](Demo/Commands%20Demo.txt) — prompts used with the `/earnings` command
- [Agent Teams Demo.txt](Demo/Agent%20Teams%20Demo.txt) — prompts used to demonstrate agent teams building a full-stack app
